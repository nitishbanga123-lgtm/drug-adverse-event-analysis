# Drug Adverse Event Analysis

## Overview
Built a pharmacovigilance analytics project using Excel, Python, SQL, and Matplotlib to evaluate adverse drug reactions, severity burden, hospitalization patterns, and comparative therapeutic safety across major drug classes.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Microsoft Excel
- Pharmacovigilance Analytics
- Data Visualization

## Key Features
- ADR Frequency by Drug
- Severity Distribution Analysis
- Hospitalization Burden by Drug
- Severe Adverse Event Profiling
- Drug Class Comparative Safety Analysis
- Executive Insights Summary

## Key Insights
- Aspirin showed the highest severe GI bleeding and hospitalization burden.
- Atorvastatin demonstrated liver toxicity and muscle-related safety concerns.
- Metformin showed frequent GI intolerance with rare severe lactic acidosis.
- Ibuprofen showed mixed mild-to-severe safety burden.
- Omeprazole had the mildest ADR profile with lowest hospitalization rates.
- Older patients displayed elevated severe adverse event susceptibility.

## Business / Clinical Outcome
This project simulated real-world pharmacovigilance and healthcare safety analytics workflows, identifying higher-risk therapies, severe ADR burden, and hospitalization trends to support healthcare intelligence and regulatory-style decision-making.
